## 💼 About the Internship
CodSoft provides virtual internship opportunities to students and tech enthusiasts. During this internship, I focused on mastering core Python concepts, logic building, data manipulation, algorithms, and interactive application design.

## 🛠️ Tasks Implemented

### 1. To-Do List Application
* **Description:** A utility application designed to help users track, manage, and prioritize daily tasks.
* **Key Features:** Add tasks, track completion status, update task descriptions, and delete tasks.

### 2. Calculator
* **Description:** A simple arithmetic tool that takes user input and performs mathematical operations.
* **Key Features:** Supports addition, subtraction, multiplication, and division with an intuitive input system.

### 3. Password Generator
* **Description:** A security-focused tool that generates strong, random passwords based on user preferences.
* **Key Features:** Customisable password length, mix of uppercase/lowercase letters, digits, and special characters.

### 4. Rock-Paper-Scissors Game
* **Description:** An interactive game played against the computer with automated rule checking.
* **Key Features:** User score tracking, random computer moves, clear winner announcements, and a replay option.

### 5. Contact Book
* **Description:** A comprehensive database-style application to store and manage contact information.
* **Key Features:** Add new contacts (Name, Phone, Email, Address), search contacts, update existing details, and delete entries.
